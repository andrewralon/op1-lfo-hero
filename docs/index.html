op1 lfo hero

custom midi lfos for the teenage engineering op-1 field

setup

  • connect your op1 via usb-c or bluetooth (ble midi)
  • for the app to drive the clock, use midi sync mode - aka app mode
  • for the op1 to drive the clock, use beat match / op1 mode - aka op1 mode
  • the app auto-detects and connects — no manual selection needed

troubleshooting

  • app shows "no device found"
    make sure the op1 is powered on and connected via usb-c (or paired via bluetooth). try a different cable — some usb-c cables are charge-only and carry no data.
  • controls send but op1 doesn't respond
    macOS coremidi can get stuck. fix: run sudo pkill -9 coremidi in terminal, then unplug and replug the op1. no restart needed.
  • tape doesn't respond to play / stop
    confirm the op1 is in midi sync mode. the BPM display should show -- while waiting for clock — if it shows an internal tempo, sync mode isn't active.
  • op1 tempo doesn't match the app
    restart the app. power cycle the op1. note: bluetooth (ble midi) cannot handle high-resolution tempo changes.
  • left / right tape buttons stop playback
    this is normal op1 behavior — receiving a song position pointer while playing causes it to pause. press play to resume.
